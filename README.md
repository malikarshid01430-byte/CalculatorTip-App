# CalculatorTip App 🧮

A simple Android application to calculate tips quickly and efficiently.

## 🚀 Features
- Tip calculation
- User-friendly interface
- Fast and accurate results

## 🛠️ Tech Stack
- Kotlin
- Android Studio
- XML Layouts

## ▶️ How to Run
1. Clone the repository
2. Open in Android Studio
3. Run the app
